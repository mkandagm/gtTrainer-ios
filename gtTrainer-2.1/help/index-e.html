gtTrainer

 

gtTrainer is Music Player with loop playback function.

You can use it by the same sense as the CD player.

You can choose music from iPod library for playing.

The Music list is save automatically, and you can be used in next time.


[Controller buttons]

 Change loop playback mode.
When turning it on, goto beginning of loop playback position.
Set beginning of loop playback.
Set ending of loop playback.
Display loop playback in/ out time.
Return to head of song, if the song play over 3 seconds
Move to previous song, if the song play under 3 seconds.
Return to head of song, if the song is first music list.
Move to next song.
If song is last of music list, then nothing.
Seeking backward.
Push when seeking then stop seeking.
If move to next or previous song then stopped seeking.
Seeking forward.
Push when seeking then stop seeking
If move to next or previous song then stopped seeking.
Start or pause playback.
Stop playback.
The music list goes to first song.
